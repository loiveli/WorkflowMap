## Options
### Git
Git is a [free and open source](https://git-scm.com/about/#free-and-open-source) distributed version control system. While Git is very popular for software development, its limitations with larger files mean that you most likely need to use Git Large File Storage (git-lfs) to store audiovisual content like 3D models or video. If you end up using Git, you still need to make a decision about which Git provider you want to use, unless you end up creating your own Git server.
#### Github
Github is probably the easiest option for Git and they provide unlimited public and private repositories for free alongside providing 10GB of LFS for free.
#### GitLab
Probably the only serious competitor to Github, they provide very similar terms as Github but with the ability to self host your own instance.

### Perforce
Perforce is practically the industry standard for AAA studios mainly due to its better support for larger files. They do offer a free tier for 5 users, but for a small studio Git should do the job just fine.
